# Українська щодня 🇺🇦

Особистий тренажёр української мови для російськомовних. Вивчай по 3 нових слова в день,
повторюй вивчене у щоденному квізі, накопичуй досвід і рівні.

> Це чисте статичне PWA — без збірки, без Node.js. Просто набір файлів.

## Возможности

- **Изучение новых слов** (3 в день): слово + перевод + примеры в контексте + озвучка, затем мини-квиз на запоминание.
- **Ежедневный квиз** по ранее выученным словам (до 10) с механикой: ошибка → 1 ретрай → если снова мимо, слово возвращается в обучение.
- **Геймификация**: XP, уровни (кап 100) и медали-лиги (🌱 🥉 🥈 🥇 💎 👑), серия дней 🔥.
- **Словарь выученного** с поиском и озвучкой, счётчик выученных слов.
- **Экспорт/импорт прогресса** в JSON (бэкап — данные хранятся в localStorage браузера).
- **Офлайн** через Service Worker, устанавливается на домашний экран как приложение.

## Запуск локально

В этой папке выполнить:

```bash
python3 -m http.server 8000
```

Открыть в браузере: <http://localhost:8000>

> Открывать именно через `http://`, а не двойным кликом по `index.html` (`file://`),
> иначе Service Worker и часть функций не заработают.

### Проверка с телефона по Wi-Fi (до деплоя)

Узнать локальный IP Mac (`System Settings → Network` или `ipconfig getifaddr en0`),
затем на телефоне в той же сети открыть `http://<IP>:8000`.

## Деплой (постоянная ссылка для телефона)

Подойдёт любой статический хостинг — собирать ничего не нужно.

**Netlify Drop (самое быстрое):** открыть <https://app.netlify.com/drop> и перетащить туда
всю папку проекта. Получите ссылку вида `https://....netlify.app`.

**GitHub Pages:** залить файлы в репозиторий → Settings → Pages → Source: ветка `main`, папка `/root`.

После деплоя открыть ссылку на телефоне → меню браузера → **«На экран Домой»** —
приложение встанет иконкой и будет работать офлайн.

## Структура

| Файл | Назначение |
|------|-----------|
| `index.html` | Точка входа (SPA) |
| `styles.css` | Тема Claude (тёплый светлый дизайн) |
| `app.js` | Вся логика: дневной цикл, квизы, XP/уровни, профиль |
| `storage.js` | Хранилище прогресса (localStorage) + экспорт/импорт |
| `data/words.js` | Встроенный словарь (uk → ru + примеры) |
| `manifest.webmanifest`, `sw.js`, `icons/` | PWA: установка и офлайн |

## Как добавить свои слова

Откройте `data/words.js` и добавьте объекты в конец массива `window.WORDS`:

```js
{ id: "w121", uk: "сонях", ru: "подсолнух", pos: "іменник", examples: [
  { uk: "Соняхи повертаються до сонця.", ru: "Подсолнухи поворачиваются к солнцу." }
]}
```

`id` должен быть уникальным. После добавления — обновите версию кеша в `sw.js`
(`ukr-daily-v1` → `ukr-daily-v2`), чтобы изменения подхватились офлайн.

## Настройки геймификации

Константы в начале `app.js`: `XP_PER_NEW_WORD`, `XP_PER_QUIZ_WORD`, `NEW_WORDS_PER_DAY`,
`DAILY_QUIZ_SIZE`, `MAX_LEVEL`, формула `xpNeededForLevel()` и пороги лиг `LEAGUES`.
